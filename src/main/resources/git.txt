将项目通过git来进行管理;
1.自己创建好基本项目结构
2.将我们的项目变成一个Git仓库 git init
3.因为我们代码仓库只需要存放我们的代码, 像class文件
编译器自身生成的文件都不需要放入到仓库中, 而且放入还会
经常出现代码冲突; (什么是代码冲突,怎么解决)
解决方案使用我们的忽略文件进行设置.gitignore  放在根目录下面(工程下面)
4.在设置好忽略文件后我们就可以将我们的代码提交到我们的本地仓库了. 这时候对于我们的文件我们需要先
通过git add 要添加的文件的路径来讲我们要放入本地仓库的代码
进行添加操作,接下来通过git commit -m "提交信息" 来真的将我们的项目提交到本地仓库, 这时候
才是真的将我们的代码保存在本地仓库了;我们希望跟别人共同进行代码的共同管理就需要将我们本地仓库
和远程git仓库进进行连接.并将代码推送到远程服务器上
这样才能跟别人进行共同开发,
git remote add origin https://github.com/JackZhangJianHai/teamProject.git
 远程服务器的地址; 这样连接好之后我们就可以进行项目从本地到服务的推送了
  git push -u origin master
